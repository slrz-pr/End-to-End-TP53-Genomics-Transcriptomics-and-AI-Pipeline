# End-to-End TP53 Genomics & AI Pipeline

## 1. Overview
This project investigates the "Guardian of the Genome" (TP53) through a multi-omic lens. It demonstrates how genomic variants disrupt transcriptomic profiles and how AI can detect these signatures to classify cancer samples.

## 2. Integrated Workflow
*   **Genomics:** Summarizing TP53 variants (Missense, Nonsense, Frameshift).
*   **Transcriptomics:** Normalizing expression data for direct downstream targets like CDKN1A.
*   **AI Classifier:** Training a Random Forest to identify malignancy signatures.
*   **Pathways:** Enriching results against the p53 Signaling Pathway.

## 3. Dataset Information
*   **Name:** Breast Invasive Carcinoma (TCGA, Cell 2015).
*   **Source:** [cBioPortal for Cancer Genomics](https://www.cbioportal.org/)[cite: 1].
*   **Access Date:** May 2026.
